# Food Delivery Backend API (FastAPI)

## Overview
This project is a backend system for a Food Delivery application built using **FastAPI**.  
It provides APIs to manage menu items, orders, cart workflow, and advanced operations like search, sorting, and pagination.

## Features

- RESTful API development using FastAPI  
- Data validation with Pydantic  
- CRUD operations (Create, Read, Update, Delete)  
- Helper functions for clean business logic  
- Cart → Order → Checkout workflow  
- Search, Sorting, and Pagination  
- Interactive API documentation with Swagger UI  

## Tech Stack

- Python  
- FastAPI  
- Pydantic  
- Uvicorn
