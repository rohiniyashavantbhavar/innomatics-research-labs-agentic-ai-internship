# FastAPI Internship Assignment

This project contains the Day-2 FastAPI practice tasks.

## Features
- Product filtering using query parameters
- Product price endpoint
- Customer feedback API with validation
- Product summary dashboard
- Bulk order processing
- Order status tracking (Bonus)

## Tech Stack
- Python
- FastAPI
- Pydantic
- Uvicorn

## Run the Project

Install dependencies:

pip install fastapi
pip install uvicorn

Run server:

uvicorn main:app --reload

Open Swagger UI:

http://127.0.0.1:8000/docs
