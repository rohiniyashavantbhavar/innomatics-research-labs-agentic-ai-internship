# GenAI Prompt Engineering Assignment

### Project: Mini Prompt Engine using LangChain  

---

## Overview
This project focuses on building a **dynamic and reusable prompt system** using LangChain instead of hardcoded prompts. It demonstrates how to design structured prompt pipelines for real-world Generative AI applications.

---

## Objectives
- Replace hardcoded prompts with **PromptTemplate**
- Build **multi-input prompt systems**
- Create **prompt variations**
- Implement **ChatPromptTemplate with roles**
- Add **input validation**
- Develop a **Prompt Generator App**
- Test **template reusability**

---

## Tech Stack
- Python  
- LangChain  
- Jupyter Notebook  

---

## Tasks Implemented

### Task 1: Replace Hardcoded Prompts
Converted static prompts into reusable `PromptTemplate`.

### Task 2: Multi-Input Prompt System
Created templates using multiple inputs like topic, audience, and tone.

### Task 3: Prompt Variations Engine
Designed different prompt styles:
- Teaching  
- Interview  
- Storytelling  

### Task 4: ChatPromptTemplate System
Implemented chat-based prompts with roles:
- Teacher  
- Interviewer  
- Motivator  

### Task 5: Input Validation
Validated inputs using predefined rules and handled invalid cases.

### Task 6: Prompt Generator App
Built a function to dynamically generate prompts based on user inputs and styles.

### Task 7: Template Reusability Test
Tested a single template with multiple inputs to ensure flexibility.

---

## Workflow

User Input → Validation → Prompt Template → Generated Prompt → Output

---

## Key Learnings
- Designing **modular and reusable prompts**
- Handling **dynamic inputs in LLM systems**
- Building **structured prompt pipelines**
- Understanding real-world **Generative AI workflows**
