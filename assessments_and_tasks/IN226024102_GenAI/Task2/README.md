# Sentiment Analysis using NLP & Machine Learning

## Project Overview

This project focuses on building a **Sentiment Analysis system** using Natural Language Processing (NLP) and Machine Learning.
The goal is to classify text data (movie reviews) into **positive** or **negative** sentiments.

---

## Dataset

* **Dataset Used:** IMDb Movie Reviews Dataset
* **Source:** Kaggle
* The dataset contains:

  * `review` → Text data
  * `sentiment` → Label (positive / negative)

---

## Technologies Used

* Python
* Pandas
* NLTK (Natural Language Processing)
* Scikit-learn

---

## Project Workflow

1. Data Loading
2. Data Understanding
3. NLP Preprocessing
4. Feature Engineering (TF-IDF)
5. Model Training
6. Model Evaluation
7. Result Comparison

---

## NLP Preprocessing Steps

* Lowercasing text
* Removing punctuation
* Removing stopwords
* Tokenization
* Lemmatization

---

## Machine Learning Models Used

* Logistic Regression
* Naive Bayes
* Decision Tree

---

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score

---

## Results

* Logistic Regression achieved the best performance
* Naive Bayes performed well for text classification
* Decision Tree showed comparatively lower accuracy

---

## Conclusion

NLP preprocessing and TF-IDF feature extraction significantly improved model performance.
Logistic Regression proved to be the most effective model for this sentiment analysis task.
